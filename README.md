# Ohitorisama

## サイト概要
写真でおいしい・楽しいを共有し、一人時間を楽しむための情報交換ツール  

### サイトテーマ
写真で探す、美味しいOhitorisama時間  

### テーマを選んだ理由
一人で気軽に食事やお酒を楽しむための情報がしたいと思い、  
ひとりで行けることに特化したサイト作成にしました。  

また、写真をメインで投稿し、どこにいくか選ぶことも楽しめる  
サイトを目指します。

### ターゲットユーザ
ひとりでも美味しいものを楽しみたい方  

### 主な利用シーン
ひとりでおいしいものが食べたいとき  
ひとりでお酒を楽しみたいとき  
一人で行きやすい店を発見し、共有したい時
